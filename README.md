
# 📊 Inventory Data Analysis Project

## 📌 Project Overview

This project focuses on analyzing inventory data for a manufacturing company to improve inventory management and optimize operational efficiency. The analysis identifies key patterns in sales, inventory levels, and vendor performance, and applies machine learning techniques for demand forecasting.

---

## 🎯 Objectives

* Determine optimal inventory levels
* Reduce stockouts and excess inventory
* Analyze inventory turnover and carrying costs
* Perform ABC analysis for product prioritization
* Forecast future demand using machine learning

---

## 🛠️ Tools & Technologies Used

* Python
* Jupyter Notebook
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn (Linear Regression)

---

## 📂 Dataset Description

The dataset consists of multiple files related to inventory operations:

* **PurchasePrice** → Product pricing details
* **BegInv** → Beginning inventory
* **EndInv** → Ending inventory
* **Invoice** → Purchase transactions
* **Final_Purchase** → Final purchase records
* **Final_Sales** → Sales transactions

---

## 🔍 Project Workflow

### 1. Data Cleaning

* Handled missing values
* Converted data types
* Removed inconsistencies

### 2. Exploratory Data Analysis (EDA)

* Analyzed top vendors
* Identified high-value products
* Visualized sales trends

### 3. ABC Analysis

* Classified products into:

  * **A** (High importance)
  * **B** (Medium importance)
  * **C** (Low importance)

### 4. Inventory Metrics

* Inventory Turnover
* Reorder Point
* Economic Order Quantity (EOQ)
* Carrying Cost

### 5. Demand Forecasting

* Built a **Linear Regression model**
* Predicted future sales demand
* Evaluated using Mean Squared Error (MSE)

---

## 📈 Key Insights

* A small number of products contribute to most revenue
* Certain vendors dominate supply chain operations
* Efficient inventory turnover improves profitability
* Forecasting helps in reducing stockouts and overstock

---

## 📊 Results

* Identified critical inventory items using ABC analysis
* Calculated optimal ordering quantities

* Provided actionable insights for better inventory control
